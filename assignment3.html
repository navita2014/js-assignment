Datatypes in Javascript

JavaScript variables can hold many data types: numbers, strings, objects and more:

JavaScript Strings:A string (or a text string) is a series of characters

JavaScript Numbers:JavaScript has only one type of numbers.Numbers can be written with, or without decimals

JavaScript Booleans:Booleans can only have two values as true or false.

JavaScript Arrays:JavaScript arrays are written with square brackets[].

JavaScript Objects:JavaScript objects are written with curly braces {}.

The typeof Operator:You can use the JavaScript typeof operator to find the type of a JavaScript variable.

Undefined:In JavaScript, a variable without a value, has the value undefined. The type is also undefined.

Null:In JavaScript null is "nothing". It is supposed to be something that doesn't exist.
